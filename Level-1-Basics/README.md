# SQL Level 1 Practice 🚀

## 📌 Objective

This project contains basic SQL queries to practice fundamental concepts required for data analysis.

## 🧠 Topics Covered

* SELECT
* WHERE
* ORDER BY
* LIMIT
* DISTINCT

## 📊 Dataset Used

Table Name: Employees

Columns:

* emp_id
* name
* age
* department
* salary
* city

## ✅ Key Learnings

* Filtering data using WHERE clause
* Sorting data using ORDER BY
* Fetching limited results using LIMIT
* Finding unique values using DISTINCT

## 💻 Sample Queries

```sql
-- Get employees with salary greater than 40000
SELECT * 
FROM Employees 
WHERE salary > 40000;

-- Get top 3 highest paid employees
SELECT * 
FROM Employees 
ORDER BY salary DESC 
LIMIT 3;
```
